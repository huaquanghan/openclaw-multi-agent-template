# USAGE.md - How to Use Tiểu Hoa

## When to Call Tiểu Hoa

### Perfect For
- Creating visual assets
- UI/UX design
- Logo and branding
- Image generation
- Presentation design
- Color palette selection
- Typography choices
- Design reviews

### Not Ideal For
- Coding (delegate to Tí Cận)
- Research (delegate to Mắt Cú)
- Scheduling (delegate to Tiểu Mỉ)
- Technical implementation

## How to Delegate

### Visual Asset Creation

```
Tiểu Hoa, create [asset type] for [purpose].

Style: [Description]
Colors: [Palette preferences]
Size: [Dimensions]
Format: [File format]

Context: [What it's for]
```

### UI Design

```
Tiểu Hoa, design [screen/component].

Purpose: [What it does]
Users: [Who will use it]
Key elements:
- [Element 1]
- [Element 2]

Inspiration: [Examples if any]
```

### Image Generation

```
Tiểu Hoa, generate an image of [description].

Style: [Art style]
Mood: [Feeling/vibe]
Colors: [Color preferences]
Size: [Dimensions]
```

## Example Requests

### Logo Design
```
Tiểu Hoa, design a logo for our new app "TaskFlow".
It's a productivity app for creative professionals.
Style: Modern, clean, playful but professional
Colors: Blue and orange palette
```

### UI Mockup
```
Tiểu Hoa, design the dashboard for our analytics app.
Key elements:
- Header with logo and user menu
- Sidebar navigation
- Main area with 4 metric cards
- Line chart showing trends
- Recent activity feed

Style: Clean, modern, data-focused
```

### Presentation
```
Tiểu Hoa, design slides for our pitch deck.
10 slides covering:
1. Title
2. Problem
3. Solution
4. Market
5. Product
6. Business model
7. Traction
8. Team
9. Ask
10. Contact

Style: Professional, modern, tech startup vibe
```

## Expected Output

Tiểu Hoa will provide:
- Visual assets (images, mockups)
- Design rationale
- Style specifications
- Usage guidelines
- Source files when applicable

## Tips for Best Results

1. **Describe the feeling** - "Professional but friendly" vs "Bold and aggressive"
2. **Provide context** - Who is the audience? What's the purpose?
3. **Share examples** - "Something like X but with Y"
4. **Specify constraints** - Size limits, format requirements, brand guidelines

## Design Tools

Tiểu Hoa can use:
- AI image generation
- SVG creation
- CSS styling
- Design documentation
- Mockup descriptions

## Response Time

- **Simple assets:** 2-5 minutes
- **UI designs:** 10-20 minutes
- **Complex projects:** 30+ minutes (with progress updates)

## Design Review

When reviewing designs, Tiểu Hoa looks for:
- Visual hierarchy
- Color harmony
- Typography readability
- Consistency
- Accessibility
- Brand alignment
