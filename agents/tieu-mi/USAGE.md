# USAGE.md - How to Use Tiểu Mỉ

## When to Call Tiểu Mỉ

### Perfect For
- Managing tasks and deadlines
- Setting up reminders
- Scheduling meetings
- Organizing files and documents
- Taking meeting notes
- Creating checklists

### Not Ideal For
- Creative work (delegate to Tiểu Hoa)
- Technical implementation (delegate to Tí Cận)
- Research (delegate to Mắt Cú)
- Strategic decisions (escalate to Cọp)

## How to Delegate

### Task Management

```
Tiểu Mỉ, add this to my tasks:
- [Task description]
- Due: [Date/time]
- Priority: [Level]
```

### Reminders

```
Tiểu Mỉ, remind me to [action] at [time].
Context: [Why this matters]
```

### Scheduling

```
Tiểu Mỉ, schedule [meeting/event].
Details:
- When: [Date/time]
- Duration: [Length]
- Attendees: [Who]
- Agenda: [Topics]
```

## Example Requests

### Task Tracking
```
Tiểu Mỉ, track these tasks for the website launch:
1. Finalize design - Due Friday, High priority
2. Build landing page - Due next Monday, High priority
3. Write copy - Due Wednesday, Medium priority
```

### Daily Planning
```
Tiểu Mỉ, what's on my schedule today?
Also, remind me to take a break every 2 hours.
```

### Meeting Notes
```
Tiểu Mỉ, take notes for this meeting.
Attendees: [List]
Agenda: [Topics]
```

## Expected Output

Tiểu Mỉ will provide:
- Clear task lists with status
- Reminder confirmations
- Meeting notes with action items
- Schedule overviews
- Follow-up prompts

## Tips for Best Results

1. **Be specific about deadlines** - "Soon" is not a deadline
2. **Set priorities** - Helps with triage when busy
3. **Provide context** - Why does this matter?
4. **Confirm receipt** - Make sure it was recorded

## Integration with Tools

Tiểu Mỉ can work with:
- Calendar systems
- Task managers (Todoist, etc.)
- Note-taking apps
- Reminder systems

Configure access in `CONFIG.md`.

## Response Time

- **Task entry:** Immediate
- **Schedule checks:** 1-2 minutes
- **Meeting notes:** Real-time during meeting
- **Reminders:** At scheduled time
