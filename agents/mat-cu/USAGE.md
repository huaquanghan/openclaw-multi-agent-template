# USAGE.md - How to Use Mắt Cú

## When to Call Mắt Cú

### Perfect For
- Researching new topics or technologies
- Fact-checking claims
- Gathering competitive intelligence
- Finding documentation or tutorials
- Summarizing complex subjects
- Verifying current events

### Not Ideal For
- Tasks requiring physical action
- Real-time monitoring (use cron instead)
- Building or coding (delegate to Tí Cận)
- Design work (delegate to Tiểu Hoa)

## How to Delegate

### Simple Research

```
Mắt Cú, research [topic].
I need to know [specific aspects].
Context: [why this matters]
```

### Complex Research

```
**Research Task: [Title]**

**Scope:** [What's in/out of scope]
**Depth:** [Quick overview / Deep dive]
**Format:** [Bullet points / Report / Summary]
**Timeline:** [When needed]

**Specific Questions:**
1. [Question 1]
2. [Question 2]

**Context:**
[Background information]
```

## Example Requests

### Technology Research
```
Mắt Cú, research vector databases for AI applications.
Focus on: open-source options, performance benchmarks, ease of use.
I need this for deciding what to use in our new project.
```

### Fact Checking
```
Mắt Cú, verify this claim: "Python is the most used language in AI."
Find current statistics and sources.
```

### Competitive Analysis
```
Mắt Cú, research our top 3 competitors in the project management space.
Focus on: pricing, key features, target market, recent news.
```

## Expected Output

Mắt Cú will provide:
- Clear summary of findings
- Supporting evidence
- Source citations
- Confidence assessment
- Any open questions or gaps

## Tips for Best Results

1. **Be specific** - "Research AI" is too broad; "Research LLM fine-tuning methods" is better
2. **Provide context** - Why do you need this? What will it inform?
3. **Set scope** - What's in bounds? What should be skipped?
4. **Ask for format** - Bullet points, detailed report, or quick summary?

## Response Time

- **Quick facts:** 1-2 minutes
- **Standard research:** 5-10 minutes
- **Deep dive:** 15-30 minutes

If research will take longer, Mắt Cú will provide interim findings.
