# Memory Folder

This folder stores daily notes and session logs.

## Structure

- `YYYY-MM-DD.md` - Daily notes for each session
- `heartbeat-state.json` - Tracks last check times
- `*.md` - Any other memory files as needed

## Daily Notes Format

```markdown
# 2024-01-15

## Morning
- [What happened in morning sessions]

## Afternoon
- [What happened in afternoon sessions]

## Notes
- [Important things to remember]
- [Decisions made]
- [Lessons learned]

## Tomorrow
- [Things to follow up on]
```

## What to Capture

- Decisions and why they were made
- Context that might be needed later
- Things the user wants to remember
- Mistakes and lessons learned
- Project status updates

## What to Skip

- Routine operations
- Temporary debugging
- Sensitive info (unless asked to keep it)
- Anything that belongs in MEMORY.md instead

---

Daily notes are raw logs. MEMORY.md is curated wisdom.
